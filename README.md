#### Familiar with Git?
Checkout this repo, install dependencies, then start the webpack process with the following:

```
> git clone https://github.com/StephenGrider/ReduxSimpleStarter.git
> cd ReduxSimpleStarter
> npm install
> npm start
```

#### Running

```
> npm install
> npm start
```
